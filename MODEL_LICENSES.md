# Model licensing and provenance

This repository's MIT licence covers only the `silero-fastapi` source code.
Downloaded Silero models retain their upstream licences.

## Modern CIS Base

`v5_cis_base_nostress` is published under the Silero CIS Base MIT licence:

- https://github.com/snakers4/silero-models/blob/master/LICENSE_CIS

The pinned model comes from:

- https://models.silero.ai/models/tts/ru/v5_cis_base_nostress.pt

## Other included catalogue models

`v5_cis_ext`, `v5_5_ru`, and the V3 international models are covered by the
main Silero repository licence, currently Creative Commons
Attribution-NonCommercial-ShareAlike 4.0 International:

- https://github.com/snakers4/silero-models/blob/master/LICENSE

The service requires an explicit acknowledgement before downloading one of
these models and exposes `commercial_use_allowed: false` in its catalogue.

Licensing can change upstream. Pandrator records the model ID, pinned SHA-256,
licence identifier, voice, and generation settings with each take. Users are
responsible for confirming that their intended use complies with the licence.
